# Part 1 Search Log (PRISMA technique) 

For each source searched, record: 
- Date:
  * of search 
- Database:
  * all databases, trail registries, and websites 
- Search Strategy/Query:
  * must be full/verbatim 
- Filters/Limits:
  * can include date ranges, languages, publication types, etc. 
- Results Count
  * AKA number of records retrieved per database 
- Notes:
